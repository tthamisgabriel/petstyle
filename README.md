Pet&Style

Página inicial fictícia de uma loja de produtos para pets com estilo desenvolvida com HTML + Tailwind CSS.

Sobre o projeto

Este projeto foi criado para praticar:

- Tailwind CSS
- Layout responsivo (mobile-first)
- Uso de Flexbox e Grid
- Organização de seções em uma landing page

Funcionalidades:
- Header com navegação
- Seção hero com destaque visual
- Listagem de produtos em grid
- Depoimentos com transição automática
- Formulário de contato
